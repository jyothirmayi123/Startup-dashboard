# 📊 Startup Analysis Dashboard

## 🚀 Overview

This project is a **Power BI dashboard** that analyzes startup funding data.
It shows funding trends, top startups, and investment patterns.

---

## 📊 Key Insights

* Total Funding: **27B**
* Avg Funding: **15.89M**
* Total Deals: **1696**

---

## 📌 Features

* 📈 Funding trend by year
* 🏆 Top funded startups
* 💼 Investment type analysis
* 📊 Interactive visuals

---

## 🛠 Tools Used

* Power BI
* DAX
* Power Query
